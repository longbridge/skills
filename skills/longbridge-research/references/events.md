## longbridge-event-opportunity — 事件驱动机会扫描

**触发场景**：_"近期有哪些并购机会"_、_"事件驱动投资机会"_（主要面向A股）

**事件类型与机会框架**：

| 事件 | 典型投资窗口 | 注意事项 |
|---|---|---|
| 被并购/私有化传闻 | 消息公布前后的套利 | 监管批准风险 |
| 实控人变更/混改 | 效率提升预期 | 整合周期长 |
| 资产注入预期 | 市值重估 | 注入资产质量 |
| 股权激励方案 | 绑定管理层利益 | 行权价格 vs 现价 |
| 定增/可转债 | 折价认购窗口 | 稀释风险 |
| 业绩反转（连续亏损→盈利） | 估值重塑 | 可持续性 |

**扫描工作流**：
1. 获取近期公司公告（`longbridge-content` 新闻子命令）
2. 识别事件类型
3. 评估事件概率和投资回报/风险比

## longbridge-event-strategy — 事件驱动策略框架

**触发场景**：制定针对特定公司事件的投资策略。

**策略设计步骤**：
1. **事件识别**：分类（并购/重组/政策/财报超预期等）
2. **概率评估**：事件发生概率（基于历史/信息分析）
3. **预期回报**：如果发生 → 预期涨跌幅；如果不发生 → 下行幅度
4. **期望值计算**：EV = P(成功) × 上涨 + P(失败) × 下跌
5. **仓位建议**：凯利公式参考：f = (P × b - (1-P)) / b

## longbridge-competitive-analysis — 竞争格局分析

**触发场景**：_"NVDA 的竞争优势"_、_"行业竞争格局"_、_"波特五力分析"_

**分析框架（Porter's Five Forces）**：

1. **行业竞争强度**：现有竞争者数量/集中度/产品差异化/价格竞争
2. **新进入者威胁**：进入壁垒（资本/技术/规模/监管）
3. **替代品威胁**：替代品可行性/转换成本
4. **供应商议价能力**：集中度/替代供应商数量
5. **买方议价能力**：客户集中度/转换成本/信息透明度

**竞争优势来源识别**：
- 成本领先（规模经济/学习曲线/独特资源）
- 差异化（品牌/技术/服务/网络效应）
- 聚焦（细分市场深耕）
