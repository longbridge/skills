# Fundamentals: Industry Analysis and Supply Chain

覆盖原技能：`longbridge-industry-overview`, `longbridge-supply-chain`

---

## longbridge-industry-overview — 行业全景报告

**触发场景**：生成某行业的全景研究报告（市场规模/竞争格局/驱动因素/风险）。

**报告结构**：
1. **行业定义与分类**：GICS/申万/MSCI 分类，边界说明
2. **市场规模**：总市场规模（TAM）、近3年增速、预测增速
3. **竞争格局**：
   - 集中度（CR3/CR5/HHI 指数）
   - 主要玩家市场份额
   - 竞争维度（价格/技术/规模/品牌）
4. **驱动因素**：需求端/供给端/政策/技术变化
5. **主要风险**：监管/技术替代/周期性/全球化
6. **投资结论**：当前周期位置 + 优选标的方向

**工作流**：
1. 获取行业内主要公司的财务数据（`longbridge-market-data` + 基本面）
2. 必要时用 WebSearch 补充宏观市场规模数据（标注来源）
3. 输出全景报告

## longbridge-supply-chain — 产业链分析

**触发场景**：某行业/公司的上中下游产业链结构分析。

**分析框架**：

| 层级 | 内容 |
|---|---|
| 上游 | 原材料/关键零部件供应商；议价能力；集中度 |
| 中游 | 核心制造/平台；技术壁垒；价值分配 |
| 下游 | 终端客户/渠道；需求弹性；替代风险 |

**分析步骤**：
1. 识别目标行业/公司在产业链中的位置
2. 用 Longbridge 行业分类 + WebSearch 梳理产业链图谱
3. 重点识别：
   - 核心瓶颈环节（利润率最高、替代难度大）
   - 风险传导路径（上游涨价→中游利润压缩）
4. 输出产业链图 + 投资方向

**举例**（半导体产业链）：
- 上游：EDA软件（Synopsys/Cadence）、硅片（信越化学）、设备（ASML）
- 中游：晶圆代工（台积电/中芯）、封装测试（ASE）
- 下游：芯片设计（NVDA/QCOM/AMD）→整机（苹果/华为）→消费者
