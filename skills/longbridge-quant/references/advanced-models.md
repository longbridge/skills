## longbridge-multifactor — 多因子截面选股

**触发场景**：_"多因子模型"_、_"因子选股"_

**标准四因子**：
| 因子 | 说明 | 典型实现 |
|---|---|---|
| 价值（Value） | 低估值 = 高回报 | BP/EP（账面市值比/盈利收益率） |
| 动量（Momentum） | 近期强者继续强 | 近12个月（排除最近1个月）收益率 |
| 质量（Quality） | 高质量公司溢价 | ROE/净利率/低负债率 |
| 规模（Size） | 小市值溢价 | 对数市值（负方向） |

**多因子合成**：
1. 各因子截面标准化（Z-score）
2. 加权合成（等权或 IC 加权）
3. 按合成得分分十组（Decile）
4. 高分组买入，低分组卖出（或多空组合）

## longbridge-factor-research — 单因子有效性研究

**触发场景**：_"BP因子有效吗"_、_"研究这个因子"_

**评估方法**：
1. **IC（信息系数）**：因子值与下期收益的Pearson相关系数；IR = mean(IC)/std(IC)
2. **RankIC**：因子秩相关（更稳健）
3. **Quintile组合**：按因子分5组，看组间收益单调性

**评判标准**：ICIR > 0.5 = 因子有效；< 0.3 = 弱效或无效

## longbridge-ml-strategy — 机器学习预测策略

**触发场景**：_"机器学习选股"_、_"预测下月收益"_

**框架**：
- **滚动窗口训练**：用前N个月数据训练，预测下1个月
- **特征工程**：技术指标（RSI/MACD）+ 基本面（PE/ROE）+ 资金流向
- **模型**：随机森林/XGBoost/LSTM（时序）
- **超参数优化**：时序交叉验证（不用前视偏差的随机CV）

**注意**：机器学习预测具有高不确定性，不可单独依赖预测信号交易

## longbridge-strategy-optimizer — 量化策略优化与回测

**触发场景**：_"帮我回测这个策略"_、_"优化参数"_

**回测流程**：
1. 定义策略规则（信号/仓位/止损）
2. 获取历史数据（K线 + 基本面）
3. 模拟执行（考虑手续费/滑点）
4. 计算绩效：年化收益/夏普/最大回撤

**过拟合防范**：样本外测试；避免在测试集上优化参数

## longbridge-volatility-strategy — 历史波动率制度策略

**触发场景**：_"波动率择时"_、_"高/低波动率制度"_

**策略逻辑**：
- 计算 HV-20（20日历史波动率）和 HV-60（60日历史波动率）
- HV-20 处于近1年低百分位（<20%）→ 低波动制度 → 趋势策略有效
- HV-20 处于近1年高百分位（>80%）→ 高波动制度 → 均值回归策略有效
- 制度切换信号：HV-20 穿越 HV-60（向上=波动率扩张；向下=收缩）
