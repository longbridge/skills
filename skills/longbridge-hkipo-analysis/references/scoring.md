Loaded by `longbridge-hkipo-analysis` SKILL.md on demand. Numeric thresholds are calibrated to HK Main Board / GEM new listings; do not edit them without updating the SKILL.md examples in lockstep.

## 四维评分框架

| 维度 | 权重 | 满分（归一化前） | 数据来源（按优先级） |
|---|---|---|---|
| 定价合理性 (Valuation) | 40% | 4.0 | `longbridge ipo detail` + `longbridge peer-comparison` + `longbridge valuation` |
| 发行质量 (Quality) | 35% | 3.5 | `longbridge ipo detail` + `longbridge news` + WebSearch（基石 / 承销商排名） |
| 市场时机 (Timing) | 20% | 2.0 | `longbridge index-quote HSI.HK` + `longbridge ipo listed` + `longbridge sector-monitor` |
| 基本面展望 (Fundamentals) | 25% | 2.5 | `longbridge ipo detail`（招股书摘要 / 财务 / 募资 / 风险 / 管理层）+ `longbridge basicinfo` + `longbridge industry-overview`；招股书更深章节通过 `longbridge ipo --help` 发现专用子命令，仍缺则 WebSearch HKEXnews |

> **归一化**：四维原始分之和满分为 12.0（4.0 + 3.5 + 2.0 + 2.5），最终对外按 10 分制输出，即 `final = raw × (10 / 12)`，保留 1 位小数。

## 维度一 — 定价合理性 (40% · 满分 4.0)

| 指标 | 计算 | 经济含义 | 强 (≥ 0.8 分) | 中 (0.4–0.8 分) | 弱 (< 0.4 分) |
|---|---|---|---|---|---|
| 发行 PE | 发行价 / 招股书披露的预测每股盈利（或近 12 个月每股盈利） | 相对同业的盈利倍数 | 较同业中位数折价 ≥ 20% | 折价 0–20% | 溢价 ≥ 0% |
| 发行 PS（成长 / 亏损公司） | 发行市值 / 近 12 个月营收 | 营收驱动公司的定价锚 | < 同业中位数 | 在中位数附近（±20%） | > 中位数 ≥ 20% |
| 发行 PB（金融 / 重资产） | 发行市值 / 净资产 | 资产质量溢价 | < 同业历史均值 | 持平 | > 均值 ≥ 10% |
| 募资规模溢价 | （发行价 − 价格区间中位）/ 价格区间宽度 | 定价激进度（招股方意愿） | 价格区间下半段 | 中位附近 | 上限定价 |

**规则：**
- 盈利公司主用 PE；亏损但有营收的成长公司主用 PS；金融 / 地产 / 重资产用 PB。
- 招股书未披露预测 EPS 时，用近 12 月口径，并在附录注明"预测口径不可得，使用 TTM"。
- 同业取 **港股同行业上市公司**，优先恒生综合行业分类；若样本 < 3 家，回退到 A 股 / 美股同业并显式标"跨市场参照"。
- 4 个子指标各 1.0 分；可不满分的子指标记 0；缺失子指标按 0 计入，并在附录注明。

## 维度二 — 发行质量 (35% · 满分 3.5)

| 指标 | 计算 | 强 (≥ 0.6 分) | 中 (0.3–0.6 分) | 弱 (< 0.3 分) |
|---|---|---|---|---|
| 基石投资者比例 | 基石认购额 / 总募资额 | ≥ 30%，且基石名单含一线机构（主权基金 / 顶级公募 / 战略产业方） | 15–30%，或基石名单含腰部机构 | < 15%，或基石全是关联方 / 名头不清的财团 |
| 国际配售超购倍数 | 国际发售认购量 / 国际发售股数 | ≥ 10× | 3–10× | < 3× |
| 公开发售超购倍数 | 公开发售认购量 / 公开发售股数 | ≥ 100×（触发最大回拨档） | 15–100× | < 15× |
| 主承销商排名 | 主承销商过去 3 年港股 IPO 承销排名（WebSearch — Dealogic / Refinitiv / HKEx 年报） | 前 10 | 前 11–25 | > 25 |
| 大股东 / 创始人锁定期 | 招股书披露的禁售期 | ≥ 180 天 | 90–180 天 | < 90 天 |

**规则：**
- 5 个子指标各 0.7 分。
- 超购倍数仅在招股截止日下午公布；若调用时尚未公布，对应子指标记为 **「等待公告」**，**不强行评分**，并在评级条注明"招股截止后可重新跑一次"。
- 基石名单"含一线机构"以招股书披露名单为准，搭配 WebSearch 核对该机构是否真实存在 + 投资记录可查；查不到的名头一律记为"信息不足"。

## 维度三 — 市场时机 (20% · 满分 2.0)

| 指标 | 计算 | 强 (≥ 0.5 分) | 中 (0.25–0.5 分) | 弱 (< 0.25 分) |
|---|---|---|---|---|
| 恒生指数近 1 月涨跌 | 招股期内 HSI vs 1 月前 | ≥ +5% | ±5% 区间 | < −5% |
| 同期港股新股首日破发率 | 近 3 个月已上市港股 IPO 中首日收盘破发数 / 总数（来源：`longbridge ipo listed`） | ≤ 20% | 20–40% | > 40% |
| 同行业近 1 月平均涨跌 | 同行业港股板块近 1 月平均涨跌（`longbridge sector-monitor` 或 `longbridge industry-overview`） | ≥ +5% | ±5% | < −5% |
| 同期港股 IPO 平均超购热度 | 近 3 个月公开发售超购倍数中位数（来源：`longbridge ipo listed`） | ≥ 30× | 5–30× | < 5× |

**规则：**
- 4 个子指标各 0.5 分。
- 同行业取招股书披露的"行业分类"，回退顺序：恒生综合分类 → GICS L2 → SIC。
- 任一指标因 CLI / WebSearch 都拿不到而完全缺失时，该子指标按 0 计入并在附录注明。

## 维度四 — 基本面展望 (25% · 满分 2.5)

按招股书章节抽取，每子项 0.36 分（共 7 项 ≈ 2.5 分），来源章节见 §Prospectus extraction：

| 子项 | 强 (≥ 0.25 分) | 中 (0.12–0.25 分) | 弱 (< 0.12 分) |
|---|---|---|---|
| 行业 TAM 与 CAGR | TAM ≥ 千亿港元等级；CAGR ≥ 15% | TAM 百亿等级 或 CAGR 8–15% | TAM < 50 亿，或 CAGR < 8% |
| 营收 3 年 CAGR | ≥ 30% | 10–30% | < 10% |
| 毛利率趋势 | 逐年改善 + 末年 ≥ 行业中位 | 平稳 ±2pp | 持续下滑 |
| 净利润 / 净亏损趋势 | 已盈利且利润率扩张 | 亏损但减亏 / 盈亏临界 | 亏损扩大且无清晰路径 |
| 募资用途中"成长性投入"比例 | 研发 + 渠道扩张 + 产能建设 ≥ 60%，偿债 ≤ 10% | 成长性 40–60%，偿债 10–30% | 成长性 < 40%，或偿债 > 30%（明确扣分） |
| 收入集中度 | 前 5 大客户 < 30% | 30–50% | > 50% |
| 管理层 / 创始人行业经验 | 核心管理层平均行业经验 ≥ 8 年 + 创始人有连续创业 / 行业资深背景 | 经验 3–8 年 | < 3 年 或 创始人无可验证背景 |

**规则：**
- 招股书未披露的子项记 0，并在附录写「招股书未披露」。
- 亏损公司须额外披露「盈利路径清晰度」（招股书"未来发展策略"章节）作为风险标签，但不进入打分；管线 / 路径不明的亏损公司，整体评级最高封顶在「中性观望」。
- 收入集中度看招股书"业务概览"中 TOP5 客户披露；只披露 TOP1 比例的，按 TOP1 × 1.5 估算 TOP5 上限并附录注明"估算"。

## 招股书章节提取 (Prospectus extraction) — 优先级 + 字段映射

按以下顺序读取招股书（先 `longbridge ipo --help` 看 `ipo` 子命令族是否覆盖；典型情况 `longbridge ipo detail <SYMBOL>` 已展开主要章节字段。`ipo detail` 没覆盖到的更深章节，再按 `--help` 选其他 `ipo` 子命令补拉，仍缺则用 WebSearch 拿 HKEXnews 原文）：

1. **行业概况 / Industry Overview** → TAM 规模、CAGR、增长驱动叙述、行业地位。
2. **财务摘要 / Financial Summary** → 近 3 年营收、毛利率、净利润 / 净亏损、经营性现金流。
3. **募资用途 / Use of Proceeds** → 各用途占比；**偿债 > 30% 必须显式扣分并标红**。
4. **未来发展策略 / Business Strategies** → 增长驱动 TOP 3、盈利路径（亏损公司必看）。
5. **风险因素 / Risk Factors** → TOP 3 实质性风险（**排除"市场风险 / 经济波动"这类通用套话**，只保留公司 / 行业 / 监管特有风险）。
6. **业务概览 / Business Overview** → 商业模式（2–3 句口语化）、TOP5 客户集中度、产品矩阵。
7. **董事及高管 / Directors & Senior Management** → 创始人 / CEO 关键背景、核心团队经验。

**规则：**
- 每个抽取出来的数字必须能在附录指回到具体章节（章节名 + 页码区间，若可得）。
- 同一数据点出现在多个章节而口径不一致时，优先采用**财务摘要**章节，并在附录注明"与 XX 章节口径差 X%"。
- 中文招股书与英文招股书并存时（H 股 + 红筹），以**中文版**为基准，引用页码标注语言版本。

## Pass / reject matrix — 最终评级

归一化后总分（满分 10）+ 关键维度门槛共同决定评级：

| 综合分 + 维度门槛 | 评级 | 输出语义 |
|---|---|---|
| 综合 ≥ 7.5 **且** 定价合理性 + 发行质量都不在「弱」 | ⭐ **推荐打新** | 综合评分高且关键维度无短板，散户可优先考虑 |
| 综合 6.0–7.5 **或** 任一关键维度（定价 / 发行质量）落在「中」 | 🟡 **中性观望** | 可少量参与（1–2 手自有资金），不建议孖展 |
| 综合 < 6.0 **或** 任一关键维度落在「弱」 | ⚠️ **建议回避** | 不建议参与；若已认购可考虑减少手数或撤单（说明在港股规则允许的撤单期内） |
| 关键数据（发行价 / 基石 / 超购倍数）≥ 2 项缺失 | ❌ **数据不足，无法评级** | 不出评级；附录列出缺失项；建议招股截止日下午重新跑 |

**护栏：**
- 任一维度落在「弱」时，评级不得为「推荐」（即使综合分过线）。
- 综合分 ≥ 8.5 但用户语境提到孖展，输出仍须包含孖展杠杆风险段。
- 拒绝原因（市场不合规 / 已上市 > 30 天 / 介绍上市）出现时直接走 Refused-Layout，不走评分。

## 申购参考 (Action frame — 仅供参考，不构成投资建议)

按评级 + 用户语境分两档输出：

| 评级 | 稳健型（自有资金） | 进取型（含杠杆，必须显式提示） |
|---|---|---|
| ⭐ 推荐 | 1–3 手 | ≤ 5 手；若用孖展，杠杆 ≤ 3 倍，破发损失会按杠杆放大 |
| 🟡 中性 | 1 手；以"试试看"心态参与 | 不建议孖展 |
| ⚠️ 回避 | 不建议参与 | 不建议参与；若已认购可考虑撤单 |
| ❌ 数据不足 | 招股截止前一日重新评估 | 同左 |

**护栏：**
- 输出中不得出现"稳赚 / 保本 / 一定大涨"。
- 不得给出具体孖展产品名 / 利率推荐。
- 必须包含一句："本参考仅基于公开数据与模型框架，散户最终决策须结合自身风险承受能力与资金使用周期。"

## 次新股模式 (场景四) 的评分调整

用户已错过申购、问"现在追合适吗"时：

- **定价合理性** 维度的对比对象从"发行价 vs 同业"改为"**当前价 vs 同业**"，并显式给出"当前价距发行价偏离 ±XX%"。
- **发行质量** 维度被替换为「**首日实际表现 + 暗盘溢价**」（来源：`longbridge ipo listed` + 暗盘 WebSearch）：
  - 首日收盘 vs 发行价偏离 ≥ +20%、暗盘溢价 ≥ +10% → 强
  - 首日 ±20%、暗盘 ±10% → 中
  - 首日 < −10% 或 暗盘破发 → 弱
- **市场时机 + 基本面展望** 维度不变。
- 输出不出"打 / 不打"评级；改为：
  - 「当前价位**已包含**多少首日溢价」（叙述性）；
  - 参考目标价**区间**（同业估值倒推：发行 PE / PS × TTM 盈利 / 营收 → 同业中位口径区间）；
  - 3 条具体风险，至少 1 条挂"流动性 / 解禁压力"。

## Risk catalogue — 必出 3 条具体风险（禁用空话）

每份报告必须从以下范畴选 3 条，**每条必须挂钩这家公司的具体数字 / 业务 / 事件**：

- **业务集中度**（TOP5 客户占比、单一产品占比、单一地区占比）
- **行业 / 监管政策**（具体监管机构 + 具体政策 + 影响场景）
- **财务 / 资本结构**（持续亏损 + 经营现金流为负、有息负债 / 净资产、研发投入与产品上市不匹配）
- **大股东 / 锁定期**（锁定期 < 180 天、大股东上市前减持记录、对赌协议）
- **同业竞争 / 替代品**（具体竞品 + 市占率口径）
- **港股流动性 / 解禁**（小盘股 + 流通比例小 + 解禁日期）
- **孖展杠杆风险**（**用户提到融资打新或评级为中以上时必出**）

**禁用：**
- 「市场风险 / 经济波动 / 政策变化」（不绑定具体事实的空话一律不收）
- 「投资有风险」（这是免责声明的事，不是风险条目）
- 「公司前景未知」（评级体系本身就在判断这件事）

## Anti-cheat reminders

- 严禁出现"稳赚 / 保本 / 一定涨 / 一定破发"等绝对化词。
- 严禁基于内幕信息 / 群消息推测（如"听说基石加码"）。
- 数据缺失就降级输出，不要用行业均值或猜测填空。
- WebSearch 数据必须带可访问的 URL；不可访问 / 不可核实的来源（如朋友圈截图）一律视为不可得。
- 不得在未提示风险的情况下默认推荐孖展打新。
- 已上市 > 30 天的标的应转交 `longbridge-stock-research`，不应在本 Skill 出"打新评级"。
- 输出语言必须与用户输入语言一致，**全文统一**，不在同一份报告中混语种。
