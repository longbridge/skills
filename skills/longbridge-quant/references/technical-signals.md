# Quant: Technical Indicator Signals

覆盖原技能：`longbridge-technical`, `longbridge-candlestick`, `longbridge-ichimoku`

---

## longbridge-technical — 核心技术指标信号引擎

**触发场景**：_"NVDA RSI超卖"_、_"MACD金叉"_、_"布林带突破"_

**支持指标**：

| 指标 | 常用参数 | 信号解读 |
|---|---|---|
| 移动均线（SMA/EMA） | 5/10/20/60/120/250日 | 金叉（短期上穿长期）=买入；死叉=卖出 |
| RSI | 14日，超买>70，超卖<30 | <30=超卖机会；>70=超买谨慎 |
| MACD | 12/26/9 | 金叉+柱状图由负转正=买入信号 |
| 布林带（Bollinger） | 20日，±2σ | 价格触下轨=超卖；触上轨=超买 |
| KDJ | 9/3/3 | K线上穿D线（金叉）=买入；KDJ>80超买 |
| OBV（成交量平衡） | - | OBV上升+价格上升=确认趋势；背离=警告 |
| ATR | 14日 | 衡量波动率；用于止损距离 |

**工作流**：
1. 获取历史K线（`longbridge-market-data` kline）
2. 在LLM中计算指标
3. 识别信号（金叉/死叉/超买超卖/背离）
4. 综合多指标共振（避免单指标假信号）

## longbridge-candlestick — K线形态识别

**触发场景**：_"NVDA出现什么K线形态"_、_"锤子线"_、_"吞没形态"_

**形态分类**：

**单根K线**：
| 形态 | 特征 | 信号 |
|---|---|---|
| 锤子（Hammer） | 下影线长≥实体2倍，小实体 | 下跌后出现=看涨反转 |
| 上吊线（Hanging Man） | 同锤子，但在上升趋势顶部 | 看跌反转 |
| 十字星（Doji） | 开收价相近，影线长 | 犹豫不决，关注后续 |
| 纺锤（Spinning Top） | 小实体，上下影线均等 | 市场犹豫 |
| 大阳线（Marubozu） | 无影线，大实体 | 强势涨/跌 |

**两根K线**：
| 形态 | 特征 | 信号 |
|---|---|---|
| 吞没（Engulfing） | 后一根K线实体完全包含前一根 | 看涨/看跌反转 |
| 孕线（Harami） | 后一根K线被前一根包含 | 趋势减速信号 |
| 穿刺线（Piercing） | 下跌后阳线收盘超过前根中点 | 看涨反转 |

**三根K线**：
| 形态 | 特征 | 信号 |
|---|---|---|
| 早晨之星（Morning Star） | 阴线+小实体+大阳线 | 看涨反转 |
| 黄昏之星（Evening Star） | 阳线+小实体+大阴线 | 看跌反转 |
| 三白兵/三黑鸦 | 三连阳/三连阴 | 趋势持续 |

## longbridge-ichimoku — 一目均衡表

**触发场景**：_"一目均衡表分析"_、_"云图"_

**五线系统**：
| 线 | 计算 | 含义 |
|---|---|---|
| 转换线（Tenkan）| (9日最高+9日最低)/2 | 短期趋势 |
| 基准线（Kijun）| (26日最高+26日最低)/2 | 中期趋势 |
| 先行带A（Senkou A）| (转换线+基准线)/2，前移26日 | 云上边界 |
| 先行带B（Senkou B）| (52日最高+52日最低)/2，前移26日 | 云下边界 |
| 延迟线（Chikou）| 当前收盘价，后移26日 | 确认信号 |

**信号解读**：
- 价格在云之上 = 强势；在云中 = 中性；在云下 = 弱势
- 转换线上穿基准线（金叉）+ 价格在云上 = 强买入信号
- 云的厚薄代表支撑/阻力强度（厚=强）
