# 海龟交易法规则速查表

## 信号判定

| 动作 | System 1（短线） | System 2（长线） |
|------|------------------|------------------|
| 做多入场 | 收盘价 > 20日最高价 | 收盘价 > 55日最高价 |
| 做空入场 | 收盘价 < 20日最低价 | 收盘价 < 55日最低价 |
| 做多出场 | 收盘价 < 10日最低价 | 收盘价 < 20日最低价 |
| 做空出场 | 收盘价 > 10日最高价 | 收盘价 > 20日最高价 |

## 核心公式

```
TR  = max(High−Low, |High−PrevClose|, |Low−PrevClose|)
N   = EMA(TR, 20)     // 初始值 = SMA(TR, 20)，此后 N = (N_prev × 19 + TR) / 20
Unit = 账户净值 × 1% ÷ (N × 股价)
止损 = 入场价 ± 2N     // 做多减，做空加
加仓 = 上次入场价 ± 0.5N  // 做多加，做空减
```

## 仓位上限

| 维度 | 上限 |
|------|------|
| 单只标的 | 4 Unit |
| 高度相关标的（同板块） | 6 Unit |
| 同方向（全部多头/空头） | 10 Unit |
| 总持仓 | 12 Unit |

## S1 例外规则

```
上次 S1 盈利出场 → 跳过本次 S1 信号
上次 S1 亏损出场 → 正常执行本次 S1 信号
无历史记录       → 正常执行
S2 不受此规则影响
```
