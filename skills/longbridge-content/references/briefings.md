## longbridge-morning-brief — 每日盘前早报

**触发场景**：_"今天早报"_、_"盘前有什么要关注"_、_"今日市场动态"_、_"昨晚美股怎么样"_

**定位**：每个交易日盘前，快速汇总隔夜要闻、市场涨跌和日内关键事件，帮助用户在开盘前掌握全局。

---

### 早报标准结构

```
📰 长桥早报 — {日期}

【隔夜行情】
• 美股：道指 {涨跌%}，纳指 {涨跌%}，标普 {涨跌%}
• 亚太：日经 {涨跌%}，恒指 {涨跌%}，A50 {涨跌%}
• 大宗商品：原油 {价格}（{涨跌%}），黄金 {价格}（{涨跌%}）
• 汇率：美元指数 {值}，人民币 {值}

【今日关键事件】
• {时间}：{事件名称}（重要性：高/中/低）
• ...

【市场要闻 Top5】
1. {标题} — {1句摘要}
2. ...

【今日财报】
• {symbol}（{公司名}）盘前/盘后

---
数据来源：Longbridge Securities
```

---

### 工作流

1. 运行 `longbridge --help` 找到市场概览和财报日历子命令
2. `longbridge <market-overview-subcommand> --format json`：获取主要指数行情
3. `longbridge <calendar-subcommand> --date today --format json`：获取今日财报/宏观事件
4. `longbridge <news-subcommand> --market all --format json`：获取市场要闻
5. 按模板组织输出；来源标注 **Longbridge Securities / 长桥证券**

### 覆盖市场

| 市场 | 内容 |
|---|---|
| 美股 | 道指/纳指/标普500/罗素2000 |
| 港股 | 恒生指数/恒科指数 |
| A股 | 上证/深证/科创50 |
| 大宗商品 | WTI原油/布伦特/黄金/铜 |
| 债券 | 美债10年期收益率 |
| 汇率 | 美元指数/USDCNH |

---

### 时区处理

- 港股/A股用户：早报对应前一交易日美股收盘行情 + 今日港股/A股盘前
- 美股用户：早报对应美东时间盘前，含隔夜期货
