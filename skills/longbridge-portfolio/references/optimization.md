# Portfolio: Optimization, Rebalancing, and Planning

覆盖原技能：`longbridge-portfolio-diagnosis`, `longbridge-portfolio-rebalance`, `longbridge-asset-allocation`, `longbridge-tax-harvesting`, `longbridge-financial-planning`

---

## longbridge-portfolio-diagnosis — 持仓诊断

**触发场景**：_"我的组合有什么问题"_、_"集中度高吗"_

**诊断维度**：
1. **集中度**：前5持仓权重（>60%=集中）；单一持仓>20%=高风险
2. **行业分布**：是否过度集中于单一行业
3. **相关性矩阵**：持仓两两相关系数（高相关=分散化效果差）
4. **地区/货币敞口**：港股/美股/A股/外汇比例

## longbridge-portfolio-rebalance — 再平衡建议

**触发场景**：_"帮我生成调仓方案"_、_"恢复目标配比"_

**再平衡流程**：
1. 获取当前持仓权重
2. 与目标权重对比，识别偏移>5%的持仓
3. 生成调仓清单（买入/卖出方向和数量）
4. 考虑：最小交易金额/手续费阈值/税务影响

**输出示例**：
```
再平衡方案 — 目标偏移阈值：5%

标的      | 当前权重 | 目标权重 | 操作 | 参考金额
NVDA.US  | 35%     | 25%     | 卖出 | -$X
AAPL.US  | 8%      | 15%     | 买入 | +$Y
现金     | 5%      | 10%     | -    | -
```

## longbridge-asset-allocation — 资产配置优化

**触发场景**：_"帮我优化资产配置"_、_"有效前沿"_

**框架**：
- **MPT（现代组合理论）**：最大化夏普比率的有效前沿
- **Black-Litterman**：在市场均衡配置基础上融入主观预期
- **风险预算**：按风险贡献（而非市值）分配权重

**简化工作流**：
1. 收集各资产类别历史收益率（5年）
2. 计算协方差矩阵
3. 求解最大夏普比率组合权重
4. 输出建议权重区间

## longbridge-tax-harvesting — 税损收割

**触发场景**：_"帮我做税损收割"_、_"哪些亏损仓位可以用来抵税"_

**分析步骤**：
1. 识别账户中的浮亏持仓（损失≥5%或绝对亏损超阈值）
2. 评估税务收益：亏损金额 × 适用税率
3. 寻找替代标的（保持相似敞口，规避 wash-sale 规则）
4. 生成操作建议（卖出亏损仓 + 买入替代品）

**注意**：美股 wash-sale 规则：卖出后30天内回购相同证券，亏损不能抵税

## longbridge-financial-planning — 个人理财规划

**触发场景**：_"退休需要多少钱"_、_"我的储蓄计划合理吗"_

**规划框架**（中国/香港/新加坡投资者）：
1. **目标设定**：退休年龄/月支出/通胀假设
2. **缺口测算**：目标资产 = 月支出 × 12 / 安全取款率（4%规则）
3. **储蓄路径**：每月定投额 = 缺口 / 复利因子（FV公式）
4. **保险需求**：重大疾病/寿险/医疗保险
5. **税务优化**：CPF/MPF/A股个人养老金账户
